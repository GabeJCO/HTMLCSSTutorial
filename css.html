CSS

CSS stands for Cascading Style Sheets. It is the language used to style the HTML elements.

CSS allows you to control the layout and appearance of your web pages. You can change the color, font, size, and spacing of your HTML elements using CSS.

CSS Syntax

The CSS syntax consists of a selector and a declaration block. The selector is used to target the HTML element, and the declaration block contains the style rules.

Here is an example of CSS syntax:

                
                    p {
                        color: red;
                        font-size: 16px;
                    }

CSS Selectors

CSS selectors are used to target HTML elements for styling. There are different types of CSS selectors, such as element selectors, class selectors, and ID selectors.

Here is an example of CSS selectors:

                
                    /* Element Selector */
                    p {
                        color: red;
                    }

                    /* Class Selector */
                    .highlight {
                        background-color: yellow;
                    }

                    /* ID Selector */
                    #header {
                        font-size: 24px;
                    }

CSS Properties

CSS properties are used to style the HTML elements. There are different CSS properties for changing the color, font, size, and spacing of the elements.

Here is an example of CSS properties:

                
                    p {
                        color: red;
                        font-size: 16px;
                        margin-top: 10px;
                    }
